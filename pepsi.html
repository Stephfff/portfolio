Development

Pepsi Stuff Canada

Pepsi Stuff Canada was a website created so you could "Buy Pepsi, Get Stuff". You would buy Pepsi products and enter your code to see if you were an instant winner. Pepsi provided the website design, and I build a working prototype out of the assets. This project included a French version of each page so I worked with a few versions of an ever-changing copy deck. The challenge for this project was to create an attractive version of the mobile site, which involved creative image carousels that would rotate through all the prizes.
